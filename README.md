## 프로젝트의 필요성
- 여름철 실외기 과열 방지
- 온습도 데이터 기반의 환경 분석 등

## 요구되는 하드웨어
- Arduino Uno
- ACS712 전류 센서
- DHT11 또는 DHT22 센서
- USB 케이블 등

## 필요한 선행 지식
- Arduino C 프로그래밍
- 시리얼 통신
- 센서 동작 원리 (전류/온습도)
